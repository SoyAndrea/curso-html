Gráficos de datos en R con ggplot2

Tipos de geometrías usadas

Geom (o combinación) Tipo de gráfico Notas / uso típico
geom_point() Gráfico de dispersión Para ver relación entre dos variables numéricas
geom_line() + geom_point() Serie temporal (línea con puntos) Permite ver continuidad + observaciones individuales
geom_col() Gráfico de barras Altura de barra proporcional al valor de la variable
geom_col() + aes(fill = categoría) Barra apilada Muestra composición interna por grupo
geom_bar() + aes(fill = categoría) Conteo de casos + barras apiladas Cuenta ocurrencias automáticamente (stat = "count")
geom_histogram() Histograma Distribución de variable numérica (por bins)
geom_boxplot() Boxplot Distribución y outliers por grupo
